How to update Website Home Banners?
Last Updated: Apr 6, 2021


  • Step 1: Go to "Documents" tab on your PringleAPI Account
  • Step 2: Click on "Choose file"
    • Note: Choose the respective file path on your computer from the drop down menu option
  • Step 3: Now click on "Upload"
    • Note: Width and Height of the Image should be 1600 X 900
    • Note: For a better performance of the website, it is advisable to upload images with smaller resolution (72 dpi)
  • Step 4: Right click and copy the link address of the image
  • Step 5: Go to "Key Value Pairs" tab
  • Step 6:Type in "Banner" in the search box
    • Note: Select the Banner that you want to update. Click on the pencil icon
  • Step 7: Paste the new link address of the image in Variable Value field
    • Note: Home page banners are numbered as Banner1, Banner2, etc
    • Note: Number of banners on homepage can be increased up to 8 banners and should be a minimum of 3 banners
    • Note: You can change them using the key value "NumberOfBanners"

How to Enable/Disable a particular secondary navigation on the website?
Last Updated: Apr 6, 2021

  • Step 1: Go to "Key Value Pairs" tab on your PringleAPI Account
  • Step 2: Type in "NavEnabled" in the search box
    • Note: NavEnabled are named as "isPromotionNavEnabled", "isMenuNavEnabled", etc
  • Step 3: Select the Nav that you want to update. Click on the pencil icon
    • Note: In place of variable value, type True / False. Eg. True(Navigation will show) / False (Navigation won’t show)
  • Step 4: Click on "Add Key Value Pair" to update
How to update the menu look & feel on website?
Last Updated: Apr 6, 2021


  • Step 1: Go to "Digital Menu" on your PringleAPI Account
  • Step 2: Go to "information"
  • Step 3: Select the template that you desire
    • Note: Tabbed Menu Template is used for Web View and Accordion Menu Template is used for mobile View
  • Step 4: Copy the URL of the selected Menu Template
  • Step 5: Go to "Key Value Pairs" tab
  • Step 6: Type in "PringleDigitalMenu" in the search box, click on the pencil icon to update, and paste the link
    • Note: You have to edit the URL
      • Remove the second half of the URL "&cartsess=jj0y0vs2ghnazxplb1x2tpeo"
      • Add &promos=1
  • Step 7: Type in "PringleDigitalMenuMobile" in the search box, click on the pencil icon to update, and paste the link
    • Note: You have to edit the URL
      • Remove the second half of the URL "&cartsess=jj0y0vs2ghnazxplb1x2tpeo"
      • Add &promos=1
  • Step 8: Click on "Add Key Value Pair"
How to update contact info like location phone & address shown on website?
Last Updated: Apr 6, 2021

  • Step 1: Go to "Key Value Pairs" Tab on your PringleAPI Account
  • Step 2: Once the page opens - in the "Search" box enter the term 'Corporate'
  • Step 3: This will show all Key Values which have the term 'Corporate'
  • Step 4: Click on the "Pencil" icon for the item you would like to update
  • Step 5: If any Corporate Key Value is missing you can add that KeyValue. Enter the "Variable Key" and "Value" into the box at the top of the page and click on "Add Key Value Pair" button
  • List of Corporate Key Value Pairs: CorporateAddressLine1, CorporateCity, CorporateCountryCode, CorporateEmail, CorporateFax, CorporateName, CorporatePostalCode, CorporateStateAbbr, Phone1
  • Note: This changes only on the website. To change your profile go to Profile on PringleAPI Account
